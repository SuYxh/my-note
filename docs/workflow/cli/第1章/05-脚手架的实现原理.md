扩展一下，有的同学可能会问下面两种写法的区别？

```bash
#!/usr/bin/env node
#!/usr/bin/node
```

- 第一种是在环境变量中查找 `node`
- 第二种是直接执行 `/usr/bin/` 目录下的 `node`
